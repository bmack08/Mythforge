    Synopsis:

    The player interacts with a console that can be used to free Shadowheart from her pod. In order to use it correctly, they must first insert an activation rune (or be a magic user who passed an earlier arcana check that provided a clue about the console), then psionically connect with the console and command it to open the pod.
