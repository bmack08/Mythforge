    Synopsis:

    Dialogue when clicking on boat in cave to reach Grymforge. Player can stay or cross. This dialogue only appears once player has already been to Grymforge.

    How to trigger:

    Repro steps (will only work with new game):
    1. Start the game.
    2. loadlevel WLD_Main_A
    3. Use debug book - go to region - act 1 - underdark - lone duergar
    4. Use the Recon command: "setOnstage S_UND_DuergarCamp_BoatAtCave"
    6. goto -16.45, 0.50, -235.73
    7. Click on the boat.
