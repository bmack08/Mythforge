    Synopsis:

    A coarse and unempathetic duergar, one of three guarding some Deep Rothe (beasts of burden currently being used to clear rubble).
    Different reactions if:
    - animals are panicking;
    - animals are calm;
    - animals are dead (doesn't matter how);

    How to trigger:

    To trigger ROTHE ARE AFRAID, NOT MET
    1. Start the game.
    2. Use Recon command: "loadlevel WLD_Main_A"
    3. goto -677.266; 13.025; 441.9 and leave the dialog with herdmaster.
    4. Talk with Morgya.

    To trigger ROTHE ARE AFRAID, MET
    1. Start the game.
    2. Use Recon command: "loadlevel WLD_Main_A"
    3. goto -677.266; 13.025; 441.9 and leave the dialog with herdmaster.
    4. Talk with Morgya.
    5. Talk to Morgya again.

    To trigger ROTHE ARE DEAD
    1. Start the game.
    2. Use Recon command: "loadlevel WLD_Main_A"
    3. goto -677.266; 13.025; 441.9 and leave the dialog with herdmaster.
    4. Use Recon command: "SetGlobalOsirisFlag UND_FearfulRothe_State_RotheAreDead"
    5. Talk with Morgya.

    To trigger ROTHE CALM, RUBBLE STILL THERE
    1. Start the game.
    2. Use Recon command: "loadlevel WLD_Main_A"
    3. goto -677.266; 13.025; 441.9 and leave the dialog with herdmaster
    4. Use Recon command: "addDebugSpell Shout_SpeakWithAnimals" and "infiniteAp"
    5. Talk with the Rothe in front of the herdmaster and convince it to attack the rubble.
    6. Rapidly talk with Morgya while rubbles are still up.
