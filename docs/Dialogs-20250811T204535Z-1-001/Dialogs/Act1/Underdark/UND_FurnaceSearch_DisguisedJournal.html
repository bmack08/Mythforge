    Synopsis:

    This book is recovered from the body of Xargrim, a wizard searching for the Adamantine Forge. It contains instructions on how to operate the forge, but is disguised as a rather smutty "nature" textbook.

    Flumph = FLUMF.

    Repro steps:
    1. Start the game.
    2. loadlevel WLD_Main_A
    3. goto 49.00, 42.15, -74.00
    4. Loot the drow's corpse
    5. Open the inventory and read the book.
