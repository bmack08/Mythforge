    Synopsis:

    Pissed off duergar raider who is standing guard while the gnomes are digging up a collapsed tunnel. He is cursing his superior, saying they should just let him suffocate. The player can make use of this and threathen the duergar, saying they will tell the sergeant unless the duergar gives them money.

    How to trigger:

    to trigger Greeting -
    1. Start the game
    2. loadlevel WLD_Main_A
    3. goto -615 14.169
    4. Talk to Kirgrath

    to trigger Mutiny happened -
    1. Start the game.
    2. loadlevel WLD_Main_A
    3. goto -615 14.169
    4. use recon command: SetGlobalOsirisFlag UND_DuergarCamp_State_Mutiny
    5. Talk toThudd

    to trigger Nere killed -
    1. Start the game.
    2. loadlevel WLD_Main_A
    3. goto -615 14.169
    4. Destroy the wall (Ctrl-Shift-K)
    5. Side with rebels.
    6. killcombat.
    7. Talk with Thudd

    To trigger Nere Killed hasmet.
    1. Start the game.
    2. loadlevel WLD_Main_A
    3. goto -615 14.169
    4. Talk toThudd
    5. Destroy the wall (Ctrl-Shift-K)
    6. Side with rebels.
    7. killcombat.
    8. Talk with Thudd
