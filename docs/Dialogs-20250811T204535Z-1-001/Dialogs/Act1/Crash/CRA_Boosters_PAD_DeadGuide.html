    Synopsis:

    PAD for when players get closer to the alternative guide's corpse at the crash site. The guide is a gith mage the replaces Lae'zel in the tutorial introduction if Lae'zel is an avatar (he knows Lae'zel). After tutorial ends and players wake up, his corpse will appear nearby (he didn't get magically saved when falling).

    How to trigger:

    1) Set flag TUT_LowerDeck_State_GuideAppeared_4d3c156f-b09c-4f18-b031-d5caa8acf184 before starting WLD_Main_A

    2) Get near WLD_Main_A 256; 0; 194


    Alternatively:

    1) Start TUT_Avernus_C as Lae'zel avatar
    2) Meet alt guide in the outside path
    3) End tutorial
    4) Get near WLD_Main_A 256; 0; 194
