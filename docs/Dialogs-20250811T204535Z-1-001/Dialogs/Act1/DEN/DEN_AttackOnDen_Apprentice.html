    Repro steps:
    1. Start the game.
    2. loadlevel WLD_Main_A
    3. oe readyattackonden
    4. Side with the tieflings and kill all goblins.
    5. goto -443.35, 1.36, -8.52
    6. Speak with Nettie.

    For other permutation you will need to:
    1. Have already met Nettie and told her about the tadpole (good and bad reult)
    2. Halsin will need to be dead.
