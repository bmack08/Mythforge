    Synopsis:

    The player is locked in Nettie's lab and has cured themselves before persuading her. Normally, script starts the dialog 'DEN_Apprentice_LockedUp' to react to this, but it can fail if the player is wild shaped for example. This node taken from the dialog 'DEN_CGB_Apprentice' gives the feedback thatNettie is going hostile.

    How to trigger:

    1. Venture forth as a Druid Human Male
    2. In recon "loadlevel WDL_Main_A"
    3. In recon "goto -437.00, 1.46, -6.00"
    4. CTRL +SHIFT+ L on Avatar to level up
    5. In recon "roll 20"
    6. Pickpocket the "Elixir of Silvanus" from Nettie to be used to cure the poison later
    7. Speak with Nettie and explain the Tadpole.
    8. Follow Nettie in other room, dialog starts
    (If Nettie triggers crime dialog, persuade her otherwise and full rest camp inside to restart Nettie's pathing)
    9. Select dialog #4 “Is everyone being captured…”
    10. Select dialog #3 “Just hurry up and cure me…”
    11. Select dialog #3 “No, nothing strange…”
    12. Select dialog "Nothing, I'm sure."
    13. Select dialog #1 “Hold out your arm”
    14. Select dialog #3 “Grab the branch from her…”
    15. Select dialog #4 "Turn and leave"

    16. Wildshape into a Wolf or any form
    17. Drink the Elixir of Silvanus
    18. Observe dialog will be triggered with an in-game view
