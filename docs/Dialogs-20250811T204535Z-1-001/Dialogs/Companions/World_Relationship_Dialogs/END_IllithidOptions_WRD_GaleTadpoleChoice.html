    Synopsis:

    The Emperor has just provided the avatar the option to transform into a mind flayer in order to use the Netherstones to defeat the Netherbrain. Gale has an interest in getting the crown, meaning he needs the stones, so transforming into a mind flayer is probably his best chance to getting it.

    How to trigger:

    Load END_Main
    debug recruit Gale
    set ORI_Gale_State_WillingForCrown
    oe end_startillithidoptions
    Talk to the Emperor by Orpheus
    Choose the option to show interest in transforming
    Dialog should break out and trigger the WRD
