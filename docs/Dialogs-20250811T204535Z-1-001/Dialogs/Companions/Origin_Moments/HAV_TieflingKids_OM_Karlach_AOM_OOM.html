    Synopsis:

    If Mattis makes it to HAV and speaks with Karlach, he will recognize her from the Blood Wars. Karlach then will give him a good advice. If companion Wyll is nearby he will join the conversation.

    How to trigger:

    1. Load SCL and set up the tiefling path using the debug book.
    2. Recruit Karlach as an avatar.
    3. In recon: goto -54; 27; 140
    4. Speak with Mattis.
    OR
    1. Load SCL and set up the tiefling path using the debug book.
    2. Recruit Karlach
    3. In recon: goto -54; 27; 140
    4. Talk to Mattis as Karlach while she is far from the party.
