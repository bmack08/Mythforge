    The guards can be convinced to let you in if you are Gith-looking, produce the box, or persuade them you have a reason to be at the creche.

    How to trigger:

    Get spotted by the main checkpoint guards. Input "oe cre_voss_friendly" and "oe cre_voss_hostile" as debug commands respectively to test the different Voss permutations ("oe cre_voss_skipped" clears both tags).
