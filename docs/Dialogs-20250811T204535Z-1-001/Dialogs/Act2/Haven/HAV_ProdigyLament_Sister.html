    Synopsis:

    Lia has been rescued from Moonrise Towers by the player and is now in Last Light. This dialogue covers multiple states: Rolan is missing, Rolan is dead, Rolan and Lia had a fight (player must pass a persuade to calm the fight, pass and fail states here), and a general state for when cal, lia and rolan are all alive and well and Lia is grateful to the player.

    How to trigger:

    Recon : loadlevel SCL_Main_A
    DebugBook : Act2 --> Tiefling Path
    DebugBook : Region --> Act2 --> Haven --> After Checkpoint
    Others : Walk to the bar
    Recon : oe rolanleaveshaven (and wait a bit - optional for dead path)
    Recon : oe prisonersreturn (and wait a bit)
    Recon : SetGlobalOsirisFlag GLO_Prodigy_State_Dead (optional for dead path)
    Others : Talk to Lia

    ---

    Resuce Cal and/or Lia from Moonrise towers (this dialogue includes solo versions for if Cal died in Moonrise), then speak to Lia in Last Light. If Rolan is present, the reunion dialogue will play first.
