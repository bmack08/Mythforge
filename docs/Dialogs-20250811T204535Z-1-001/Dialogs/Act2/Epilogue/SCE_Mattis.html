    Synopsis:

    Mattis is quietly worried by Mol's disappearance but is trying to put a brave face on. In the absence of Alfira and Cerys, he is preparing to lead the tieflings to Baldur's Gate himself.

    How to trigger:

    1. Run oe SCE_TieflingFollowUp_SetFlags
    2. Run:
    oe SCE_TieflingFollowUp_AllTieflings
    -or-
    oe SCE_TieflingFollowUp_NoCerys
    -or-
    oe SCE_TieflingFollowUp_NoAlfira
    3. Run oe SCE_SetupDebrief
    4. Click through the Jaheira/Florrick dialogue

    Add eyepatch with: oe SCE_TieflingFollowUp_AddEyepatch
