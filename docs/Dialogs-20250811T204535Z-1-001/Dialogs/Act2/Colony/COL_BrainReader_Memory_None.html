    Synopsis:

    The default dialog that plays when interacting with the brain reading device/severed head in the lower Colony. Interacting with the severed head will yield no information, as it only serves to narrate memories from brain jars inserted into the device.

    How to trigger:

    - Load up GustavDev/SCL_Main_A
    - Use ReCon command "oe MemLib_AllJars" to get all the brain jars and teleport to the Brain Reading Device.
    - In front of you will be a severed head. Talk to it without inserting a brain jar into the receptacle next to it, and it will give this dialogue.
