    Dialog featuring a goblin and half-orc. Triggers the start of the caravan escort with the Drider appearing.

    The Drider is male, half-drow elf and half spider (bottom half). He carries a magical lantern to ward off the shadows

    - go to Region: SCL -> Mountain Pass
    - set Act 2 Absolute Path
    - goto 92.91, 51.44, -50.63
    - talk to Kansif, perform the Lyre
    - talk to Drider
