    Synopsis:

    Femme Harper Ranger after the fight with the Drider. To ppath: You started the fight with the drider, but switched sides. If you don't give the Harpers the moonlantern they came for, you will need to fight them for it.

    How to trigger:

    From tutorial
    Debug book → Go to Region → Act 2 → SCL → Entry Point Mountain Pass
    Debug book → Act 2 Setup → Absolute path: gives Spider's lyre to unlock the Drider's path.
    in reCon: teleport S_SCL_DriderCaravan_HalfOrcCaster
    Say you are true soul
    talk to him again → play lyre to draw in drider
    talk to drider → choose to go to moonrise → follow drider to ambush point
    In ambush dialogue → turn on drider and side with harpers
    In combat → ctrl k all harpers first except S_SCL_Drider_Harper_Ranger_001 (Harper Skywin) and then drider
    Talk to Harper Skywin
