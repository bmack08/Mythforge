    Synopsis:

    Dialog with Oliver, in which the player confronts him and convinces him to rejoin Thaniel, his other half. so that the shadow curse can be lifted. Oliver will always return to Thaniel after this dialogue, but the player can be harsh or gentle in how they do so.

    How to trigger:

    1. Debug Book - Set Quest Flags - Act 2 - GLO Lifting the Curse - Halsin at camp with Spirit
    2. Debug Book - Set Quest Flags - Act 2 - GLO Lifting the Curse - Oliver fled to Town - this will teleport you to the plaze in the Town
    3. Approach Oliver, he will start a fight and summon some creatures.
    4. Teleport closer to him - he'll unsummon the creatures and start this dialog with you.
