    Synopsis:

    SwD dialog for dead True Soul that died of hunger/thirst while hiding from Shadow Mastiffs in the island of light. They used Continual Flame spell to make torches around them ever-burning, which allowed them to hide there indefinitely (until they died). The kicker - description of the spell says that each casting uses Ruby Dust worth 50gp, so, presumably, that's one rich True Soul.
