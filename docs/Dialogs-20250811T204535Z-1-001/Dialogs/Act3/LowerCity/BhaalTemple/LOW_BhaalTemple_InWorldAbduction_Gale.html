    Synopsis:

    In certain areas in the Undercity, Orin will abduct and impersonate a target companion. Choice of target is the same as the camp abduction, but method and impersonation is not tied to the camp.

    How to trigger:

    To trigger the In-World Abductions
    1) CTRL+SHIFT+9 (Debug book) Go to Region -> Act 3 -> Undercity -> Bhaal Temple -> In-World Abdcution 1.
    2) CTRL+SHIFT+8 (Camp debug book) - Invite Individual -> Gale
    3) Walk forward a bit down the stairs and you should trigger the event. Orin-as-Gale will run towards you and start the dialog.
    4) If you want to try the other abduction location, you'll need to reload and use debug book to go to the other 3 locations.
