    Synopsis:

    Shadowheart's evil ending in which she executes clerics of Selune and Shar and becomes the prophet / founder of a new faith.

    How to trigger:

    -> Start game as Avatar Shadowheart
    -> Ctrl + 9 to Open Debug book
    -> Quests and Flags
    -> Act 3 (END)
    -> Master End Setup
    -> Agree to go to endgame
    -> Go back to master set up as above
    -> Skip Party configuration and do setup
    -> Emperor is mindflayer with stones
    -> Skip through following few options
    -> Choose to go to Defeat of Brain
    -> The Final decision scene will trigger
    -> Select the option to betray emperor and become the absolute
    -> Skip through rest of the scene into evil avatar scene
    -> Skip through until first question hold
    -> Chose fist option 'For too long, you allowed yourself...
    -> Evil Shadowheart will start here
