    Synopsis:

    This PAD is played when the player returns to the circus, but it's gone. It could be expected - because the player had completed the Dribbles quest, or if the Rinmaster was killed, so the phrase and emotion should stay vague.

    How to trigger:

    BGO_Main_A,
    Make circus disappear:
    a) complete Dribbles quest,
    b) kill Ringmaster and long rest,
    c) start combat with the clown, leave it and long rest
    d) oe WYR_Circus_TestDisappear
    Return.
