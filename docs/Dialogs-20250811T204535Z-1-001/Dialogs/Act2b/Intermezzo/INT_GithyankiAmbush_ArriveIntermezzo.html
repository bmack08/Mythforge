    Players are shown to arrive in the Intermezzo camp.

    How to trigger:

    Leave SCL towards BGO.

    For quick testing, in builds use the DebugBook -> Go to region -> Act2b - Intermezzo

    In the editor, Load INT_Main_A and start playing, you would automatically see the dialog play.

    If you want to test repeatedly in editor, start playing, press CTRL + Shift + 8 to open the Camp Debug Book.Then navigate to Queue Camp Night and select ArriveIntermezzo
