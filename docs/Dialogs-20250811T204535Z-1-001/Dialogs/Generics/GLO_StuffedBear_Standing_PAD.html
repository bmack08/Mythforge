    Synopsis:

    Launched when player interacts with an item with one of the following root templates.
    DEC_GEN_StuffedBear_Standing_A_f8d74269-6a5c-461e-8907-2e1a7b18e1ca
    DEC_GEN_StuffedBear_Standing_B_8177353f-5beb-4120-85ec-a4fa261795ae
    DEC_GEN_StuffedBear_Standing_C_d760ca77-927c-4880-98a2-991926cdd2a3

    First a skill check roll is done, then the AD is launched. Global flag GLO_TemplateUseSkillCheck_StuffedBear_Success can be used to check if the roll was successful

    How to trigger:

    Teleport to any of the stuffedbear_standing in the recon command and click on it.
